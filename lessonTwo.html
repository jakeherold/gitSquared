Git Status

From time to time, you'll find yourself with no idea where you're at in the Git process. While some folks would go to Github to look through their commits, or hammer random commands into the terminal, you don't think those methods are for you. So, whats the play, then?

Simple. We use the command "git status" to find out where we're at in the process. It'll tell us if we have files have been added or commited, and which ones have yet to have that happen. Its a tool that you won't use every time you add things to Github, but its one that you won't want to be without in the long run.
